# 服装/鞋包挂售场景指南

## 季节推断

| 商品类型 | 适用季节 |
|----------|----------|
| 夹克、风衣、卫衣、针织衫 | 春秋季 |
| 羽绒服、棉衣、毛呢大衣 | 秋冬季 |
| T恤、短裤、背心、凉鞋 | 夏季 |
| 基础款衬衫、休闲裤 | 四季 |

## 图片识别字段

| 字段 | 可识别性 | 说明 |
|------|---------|------|
| 品牌 | ✅ | 从 Logo/标签读取，调 brands 搜索确认 |
| 款式 | ✅ | 从整体外观判断 |
| 成色 | ✅ | 根据磨损/污渍/起球程度判断 |
| 尺码 | ⚠️ | 从标签/吊牌/水洗标读取 |
| 货号 | ⚠️ | 从标签/吊牌读取 |
| 颜色/材质 | ⚠️ | 能识别则识别，否则汇总时问 |

## 品牌搜索建议

- 奢侈品（LV、Gucci、Chanel 等）：搜完整英文名
- 运动品牌（Nike、Adidas 等）：搜英文名，小写即可
- 国内品牌：搜中文名
